embed a multi template

Here is an example of an embedded object which may be of different types.

The property __type__ defines which template is used.

This is a callout

Calling out to the world !!!!

  • render context
  • templates
{
 "embedded": {
  "level": "info",
  "headline": "This is a callout",
  "text": "Calling out to the world !!!!",
  "__type__": "/callout"
 },
 "name": "embed a multi template",
 "index": 4,
 "template": "multi-template-example",
 "path": "/tutorial/advanced templating/embed a multi template",
 "parent": "/tutorial/advanced templating"
}
{{#embedded}}
{{> (lookup . '__type__')}}
{{/embedded}}
/callout 
<div class="callout {{level}}">
    <h4>{{headline}}</h4>
    <p>{{{text}}}</p>
</div>
/alert 
<div class="alert alert-{{level}}">{{{text}}}</div>
/paragraph 
<h1>{{{headline}}}</h1>
<p>{{{text}}}</p>
