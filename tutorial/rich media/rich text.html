rich text

The richtext editor support WYSIWYG text editing features. It also supports links and images.

Caution: 

Double-clicking a link in the richtext editor will open the link dialog without a url. Always select the link first and then click the link icon in the toolbar.

Images are currently not properly rendered in the richtext editor but only in the generated html.

Rich Documentation can include links. 


And formatted text:
  • bold
  • italic
  • underlined
  • strikethrough
Indentation is achieved using a 40px margin with blockquote:

lorem ipsum dolor. asdasd
sad asdasdasd
asd
