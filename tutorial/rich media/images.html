images

Images are stored externally or in the file store. To use an image for from the file store you need to use the image-src helper.


to an attribute for an image use the file-ref attribute. To restrict the access to only a certain set of files set the query expression. In this case it is set to 'images/*${0}*', so the author can only choose files from the images directory.
