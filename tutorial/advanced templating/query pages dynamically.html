query pages dynamically.

A query-ref attribute supports getting the data of other pages based on a query. This can be useful to create navigations based on the page hierarchy.

The query can either be static or it can be generated by a javascript snippet. The query is mongodb-like in that it supports $regex and $in clauses. In this case the query script looks for pages whose path contain the given term:

var term = this.getValue("/term");
returnValue = {path:{$regex:term}};
