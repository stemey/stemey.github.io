outer example

To have a common header and footer for a page you can use a template-ref with the property outer set to true. Uncheck the general value to see the effect of removing the outer.


This can only be used once per page because the page's template is renamed to inner and rendered in the context of the referenced template as a partial. This limitation will be lifted soon.

There for the outer template must include this snippet:

..
{{#inner}}
{{>inner}}
{{/inner}}
..
