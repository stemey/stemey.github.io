embed a page as partial

It is also possible to embed the referenced page's template as a partial and add the data to the current render context.

This can be useful when rendering a navigation. It is the same on all pages but the highlighted item is passed from the parent context to the navigation.

Go get the new bathtub! only 3 € today!

  • templates
{
 "page": {
  "name": "An Ad",
  "template": "/Ad.json",
  "text": "Go get the new <strong>bathtub</strong>!",
  "path": "/components/An Ad.json",
  "parent": "/components"
 },
 "extra": "  only 3 € today!",
 "name": "embed a page as partial",
 "index": 1,
 "template": "/embed-partial-example.json",
 "path": "/tutorial/advanced templating/embed a page as partial.json",
 "parent": "/tutorial/advanced templating"
}
{{#page}}
{{>page extra=../extra}}
{{/page}}
page 
<span class="well">{{{text}}}<strong>{{extra}}</strong></span>
