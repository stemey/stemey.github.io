lots of multi templates

To embed a dynamic number of content elements with different templates a primitive array of multi-template-ref is employed.


The handlebars template iterates over the array an includes a dynamical partial. The name of the partial is looked up in the variable __type__.

Layout
In this case the templates are all in a verticalgroup layout which means that their size needs to be defined by their container. Hence it is a good idea to use a tree container.

{
 "embedded": [
  {
   "level": "danger",
   "text": "RED ALERT !!!",
   "__type__": "/alert.json"
  },
  {
   "level": "warning",
   "headline": "Hello",
   "text": "We are informing you with a callout.",
   "__type__": "/callout.json"
  },
  {
   "text": "Just some <b>content</b>.",
   "__type__": "/paragraph.json"
  }
 ],
 "name": "lots of multi templates",
 "index": 5,
 "template": "/multi-array-template-example.json",
 "path": "/tutorial/advanced templating/lots of multi templates.json",
 "parent": "/tutorial/advanced templating"
}
<ul>
{{#embedded}}
    <li>
    {{> (lookup . '__type__')}}
    </li>
{{/embedded}}
</ul>
/alert.json 
<div class="alert alert-{{level}}">{{{text}}}</div>
/paragraph.json 
<h1>{{{headline}}}</h1>
<p>{{{text}}}</p>
/callout.json 
<div class="callout {{level}}">
    <h4>{{headline}}</h4>
    <p>{{{text}}}</p>
</div>
